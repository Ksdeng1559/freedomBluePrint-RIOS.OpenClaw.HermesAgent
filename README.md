# Freedom Blueprint RIOS OpenClaw HermesAgent Operating System

This repository converts Matt Ganzak's Freedom Blueprint into an operator-grade business operating system for building, selling, productizing, and scaling AI-powered business systems.

The original Freedom Blueprint teaches the path from Claude Code learning to first clients, then productized services, then SaaS. This repository adapts that path into a RIOS-style execution system using:

- Claude / ChatGPT for strategy and PRD planning
- Claude Code / Codex for technical execution
- OpenClaw-style memory, outreach, and agent workflows
- Hermes Agent as the long-running orchestration layer
- Supabase / GitHub / Vercel / Railway as deployment infrastructure
- Human-in-the-loop review for every client-facing or financial decision

## Operating Doctrine

Build revenue infrastructure, not random apps.

The system follows this loop:

1. Identify a painful business workflow
2. Capture the requirement as a PRD
3. Build the smallest sellable tool
4. Deploy it live
5. Sell it with a demo and outcome case
6. Convert build revenue into support retainers
7. Productize the repeatable offer
8. Turn repeated client pain into SaaS

## First Use

Start here:

1. Read `00-START-HERE/OPERATING-MANUAL.md`
2. Choose one offer from `05-OFFER-LIBRARY/OFFER-MENU.md`
3. Use `03-PRD-FACTORY/PRD-INTERVIEW-PROMPT.md` to create a PRD
4. Execute using `04-BUILD-LAB/CLAUDE-CODE-BUILD-LOOP.md`
5. Sell using `06-OUTREACH-ENGINE/FIRST-25-OUTREACH-SOP.md`

## Strategic Positioning

This is not a beginner coding course. It is a business operating system for becoming a GTM Engineer / AI Systems Builder.

The positioning:

> We design and deploy AI-powered revenue infrastructure for businesses that need faster lead capture, qualification, follow-up, operations, reporting, and decision support.

## Core Rule

No build starts without a PRD.

No client-facing system ships without QA.

No automation sends outbound messages, spends money, touches sensitive data, or makes legal/financial decisions without human approval.
